SmallWordpressSlideshow
=======================

Small Wordpress theme function.php slideshow

Demo: http://razko.no/slideshow/

Use shortcode in posts/pages:

[slideshow ids="a,b,c"]

Outputs:
```html
<div class="slideshow">
	<ul>
		<li><img src="img.jpg" alt=""></li>
		<li><img src="img.jpg" alt=""></li>
		<li><img src="img.jpg" alt=""></li>
	</ul>
	<span class="prev">&#8249;</span>
	<span class="next">&#8250;</span>
</div>
```
